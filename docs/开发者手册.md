# RadishMD 开发者手册

## 简介

本手册面向 RadishMD 项目的开发者和贡献者，介绍项目开发环境搭建、代码结构、打包部署等相关内容。

---

## 环境要求

### 通用要求

- Node.js 18+
- Rust 1.70+
- npm 或 pnpm

### Windows 开发

- Windows 10/11
- Visual Studio Build Tools (MSVC)
- Rust MSVC 目标

### Linux 开发

- Ubuntu 20.04+ 或其他主流 Linux 发行版
- build-essential
- libwebkit2gtk-4.1-dev
- libssl-dev

### macOS 开发

- macOS 10.15+
- Xcode Command Line Tools
- Rust macOS 目标

---

## 项目结构

```
/src
  App.tsx                    # 主应用组件
  main.tsx                   # React 入口
  index.css                  # Tailwind CSS + 主题变量
  /components
    MilkdownEditor.tsx      # Milkdown 编辑器封装
    Toolbar.tsx              # 工具栏菜单
    /ui                      # shadcn-ui 组件
      button.tsx
      separator.tsx
      dropdown-menu.tsx
      dialog.tsx
  /lib
    utils.ts                 # 工具函数
/src-tauri
  src/main.rs               # Rust 入口
  src/lib.rs                # 插件初始化
  Cargo.toml                # Rust 依赖
  tauri.conf.json           # Tauri 配置
  capabilities/             # 插件权限配置
/public
  title.svg                # 应用图标
```

## 开发环境搭建

### 1. 克隆项目

```bash
git clone <repository-url>
cd RadishMD
```

### 2. 安装前端依赖

```bash
npm install
```

### 3. 配置 Rust 环境

#### Windows

```bash
# 安装 Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 添加 MSVC 目标
rustup target add x86_64-pc-windows-msvc
```

#### Linux

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install build-essential libwebkit2gtk-4.1-dev libssl-dev

# 安装 Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

#### macOS

```bash
# 安装 Xcode Command Line Tools
xcode-select --install

# 安装 Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### 4. 运行开发模式

#### Windows

```bash
powershell.exe -ExecutionPolicy Bypass -File run_dev.ps1
```

或直接运行：

```bash
npm run tauri dev
```

#### Linux / macOS

```bash
npm run tauri dev
```

首次运行需要编译 Rust 代码，可能需要 10-20 分钟。

## 构建与打包

### 构建前端

```bash
npm run build
```

### 构建 Tauri 应用

```bash
npm run tauri build
```

构建完成后，安装包位于：

- Windows: `src-tauri/target/release/bundle/nsis/`
- Linux: `src-tauri/target/release/bundle/deb/` 或 `AppImage`
- macOS: `src-tauri/target/release/bundle/dmg/`

---

## Windows 打包

### 方法一：使用 npm 命令

```bash
npm run tauri build
```

这将生成：

- `RadishMD_x.x.x_x64-setup.exe` (NSIS 安装程序)
- `RadishMD_x.x.x_x64.msi` (MSI 安装包，可选)

### 方法二：单独构建

```bash
# 进入 Tauri 目录
cd src-tauri

# 构建发布版本
cargo build --release

# 打包成安装程序
cargo tauri build
```

### Windows 打包注意事项

1. 确保已安装 Visual Studio Build Tools
2. 需要添加 MSVC 目标：`rustup target add x86_64-pc-windows-msvc`
3. 图标文件位于 `src-tauri/icons/`
4. 配置文件位于 `tauri.conf.json`

## Linux 打包

### 安装 Linux 构建依赖

#### Ubuntu/Debian

```bash
sudo apt update
sudo apt install build-essential libwebkit2gtk-4.1-dev libssl-dev curl wget file
```

#### Fedora

```bash
sudo dnf install gcc gcc-c++ make openssl-devel webkit2gtk4.1-devel
```

#### Arch Linux

```bash
sudo pacman -S base-devel openssl webkit2gtk-4.1
```

### 构建 Linux 应用

```bash
npm run tauri build -- --target x86_64-unknown-linux-gnu
```

### 打包格式

#### Debian/Ubuntu (.deb)

```bash
cargo tauri build --bundles deb
```

#### AppImage

```bash
cargo tauri build --bundles appimage
```

#### RPM

```bash
cargo tauri build --bundles rpm
```

### Linux 打包注意事项

1. WebView2 依赖：大多数现代 Linux 发行版已预装
2. 确保在 Docker 或虚拟机中交叉编译，以生成兼容更多发行版的二进制文件
3. 使用 `tauri.conf.json` 中的 `bundle.linux.deb` 配置自定义包信息

## macOS 打包

### 环境要求

- macOS 10.15+
- Xcode 15+
- Apple Developer 证书（用于签名）

### 构建 macOS 应用

```bash
npm run tauri build
```

### 打包格式

#### .dmg (磁盘镜像)

```bash
cargo tauri build --bundles dmg
```

#### .app (应用Bundle)

```bash
cargo tauri build --bundles app
```

### 代码签名与公证

#### 开发证书签名

```bash
# 签名应用
codesign --sign "Developer ID Application: Your Name" --timestamp "http://timestamp.apple.com/ts01" --options runtime ./path/to/app

# 验证签名
codesign --verify --verbose=2 ./path/to/app
```

#### App Store 分发

```bash
# 需要 Apple ID 和 App Store Connect 证书
cargo tauri build --bundles app
```

#### 公证 (Notarization)

```bash
# 上传进行公证
xcrun notarytool submit ./path/to/app.dmg --apple-id "your@email.com" --password "app-specific-password" --team-id "YOUR_TEAM_ID"

# 等待公证完成
xcrun notarytool wait ./path/to/app.dmg --apple-id "your@email.com" --password "app-specific-password" --team-id "YOUR_TEAM_ID"
```

### macOS 打包注意事项

1. 需要 Apple Developer 账户才能进行代码签名和公证
2. 图标需要 .icns 格式
3. 使用 `tauri.conf.json` 中的 `bundle.macOS` 配置
4. Rust 目标：`rustup target add x86_64-apple-darwin aarch64-apple-darwin`

## 跨平台构建

### 使用 Docker 进行 Linux 交叉编译

创建 Dockerfile：

```dockerfile
FROM ubuntu:22.04

RUN apt update && apt install -y \
    build-essential \
    curl \
    wget \
    file \
    libwebkit2gtk-4.1-dev \
    libssl-dev \
    pkg-config \
    libayatana-appindicator3-dev

RUN curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y
ENV PATH="/root/.cargo/bin:${PATH}"

WORKDIR /app
COPY . .
RUN npm install && npm run build
RUN cargo build --release --manifest-path src-tauri/Cargo.toml --target x86_64-unknown-linux-gnu
```

构建：

```bash
docker build -t radishmd-builder .
docker run --rm -v $(pwd):/app/output radishmd-builder \
    cp src-tauri/target/x86_64-unknown-linux-gnu/release/radishmd /app/output/
```

## 配置说明

### tauri.conf.json

主要配置项：

```json
{
  "productName": "RadishMD",
  "version": "0.1.0",
  "identifier": "com.radishmd.app",
  "app": {
    "windows": [{
      "title": "RadishMD",
      "width": 800,
      "height": 600
    }]
  },
  "bundle": {
    "active": true,
    "targets": "all",
    "icon": [
      "icons/32x32.png",
      "icons/128x128.png",
      "icons/128x128@2x.png",
      "icons/icon.icns",
      "icons/icon.ico"
    ]
  }
}
```

### 更换图标

1. 准备 SVG 或 PNG 图标（建议 512x512 以上）
2. 运行图标生成命令：

```bash
npx tauri icon public/your-icon.svg -o src-tauri/icons
```

支持的输入格式：SVG、PNG

## 常见问题

### Q: Windows 构建失败？

1. 检查 Rust 和 MSVC 目标是否正确安装
2. 确保 Visual Studio Build Tools 已安装

### Q: Linux 构建失败？

1. 检查系统依赖是否完整安装
2. 尝试更新 Rust：`rustup update`

### Q: macOS 签名失败？

1. 检查证书是否有效
2. 确保钥匙串访问已授权

### Q: 应用图标不显示？

1. 重新生成图标：`npx tauri icon public/title.svg`
2. 清除缓存并重新构建
