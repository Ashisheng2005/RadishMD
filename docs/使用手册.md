# Markdown Editor 使用手册

## 简介

Markdown Editor 是一款基于 Tauri + React 构建的桌面端 Markdown 编辑器，类似于 Typora 的所见即所得（WYSIWYG）编辑体验。

## 功能特性

- **所见即所得编辑** - 实时预览 Markdown 渲染效果
- **GitHub 风格 Markdown (GFM)** - 支持表格、任务列表、代码块等
- **文件操作** - 打开和保存 Markdown 文件
- **主题切换** - 支持深色/浅色主题
- **跨平台** - 基于 Tauri，性能优异，体积小巧

## 部署方法

### 环境要求

- Node.js 18+
- Rust 1.70+
- Windows 10/11 (或其他支持 Tauri 的系统)

### 安装步骤

1. **克隆或下载项目源码**

2. **安装依赖**
   
   ```bash
   npm install
   ```

3. **运行开发模式**
   
   ```bash
   # Windows
   powershell.exe -ExecutionPolicy Bypass -File run_dev.ps1
   
   # 或使用 npm (需要先配置 Rust 环境)
   npm run tauri dev
   ```

4. **构建生产版本**
   
   ```bash
   npm run tauri build
   ```

构建完成后，安装包位于：`src-tauri/target/release/bundle/`

## 使用方法

### 打开文件

1. 点击工具栏中的 **Open** 按钮
2. 在弹出的文件对话框中选择 Markdown 文件（.md, .markdown, .txt）
3. 文件内容将自动加载到编辑器中

### 保存文件

1. 点击工具栏中的 **Save** 按钮
2. 在弹出的保存对话框中选择保存位置
3. 输入文件名（建议使用 .md 扩展名）
4. 点击保存即可

### 主题切换

点击工具栏中的 **Light/Dark** 按钮可以在深色和浅色主题之间切换。

### 快捷键

编辑器支持以下快捷键：

| 功能  | 快捷键              |
| --- | ---------------- |
| 撤销  | Ctrl + Z         |
| 重做  | Ctrl + Shift + Z |
| 复制  | Ctrl + C         |
| 粘贴  | Ctrl + V         |
| 剪切  | Ctrl + X         |
| 保存  | Ctrl + S         |

## 支持的 Markdown 语法

### 基本语法

- **粗体**：`**粗体**` 或 `__粗体__`
- *斜体*：`*斜体*` 或 `_斜体_`
- ~~删除线~~：`~~删除线~~`
- `行内代码`：`` `代码` ``

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 列表

无序列表：

- 项目 1
- 项目 2

有序列表：

1. 第一项
2. 第二项

任务列表：

- [x] 已完成
- [ ] 待完成

### 代码块

```javascript
function hello() {
  console.log("Hello World!");
}
```

### 表格

| 标题1 | 标题2 | 标题3 |
| --- | --- | --- |
| 内容1 | 内容2 | 内容3 |

### 引用

> 这是一段引用文本

## 常见问题

### Q: 打不开编辑器？

确保已正确安装 Rust 工具链和 Visual Studio Build Tools。

### Q: 文件保存失败？

检查目标文件夹是否有写入权限。

### Q: 中文显示异常？

编辑器支持 Unicode，請确保系统安装了相应的字体。
