# RadishMD 使用手册

## 简介

RadishMD 是一款基于 Tauri + React 构建的桌面端 Markdown 编辑器，提供类似于 Typora 的所见即所得（WYSIWYG）编辑体验。

## 功能特性

- **所见即所得编辑** - 实时预览 Markdown 渲染效果
- **GitHub 风格 Markdown (GFM)** - 支持表格、任务列表、代码块等
- **文件操作** - 打开和保存 Markdown 文件
- **主题切换** - 支持 4 种主题：浅色、Nord、深色（VS Code）、暖色（Typora）
- **代码块语言选择** - 支持搜索和选择 100+ 编程语言
- **跨平台** - 基于 Tauri，性能优异，体积小巧

## 界面介绍

### 菜单栏

顶部为菜单栏，包含以下菜单：

- **文件** - 新建、打开、保存、另存为、关闭

中间为编辑区域，支持所见即所得的 Markdown 编辑。

## 使用方法

### 新建文件

1. 点击菜单 **文件 -> 新建** 或使用快捷键 `Ctrl + N`
2. 如果当前文档有修改，系统会提示保存

### 打开文件

1. 点击菜单 **文件 -> 打开** 或使用快捷键 `Ctrl + O`
2. 在弹出的文件对话框中选择 Markdown 文件（.md, .markdown, .txt）
3. 文件内容将自动加载到编辑器中

### 保存文件

1. 点击菜单 **文件 -> 保存** 或使用快捷键 `Ctrl + S`
2. 如果是新建文件，会弹出保存对话框
3. 选择保存位置，输入文件名（建议使用 .md 扩展名）
4. 点击保存即可

### 另存为

1. 点击菜单 **文件 -> 另存为** 或使用快捷键 `Ctrl + Shift + S`
2. 在弹出的对话框中选择新的保存位置

### 主题切换

点击菜单 **主题**，选择所需主题：

- **浅色** - 白色背景，适合光线充足环境
- **Nord** - 蓝灰色调，简洁优雅
- **深色** - 深灰背景，类 VS Code 风格
- **暖色** - 暖灰背景，类 Typora 风格

## 支持的 Markdown 语法

### 基本语法

- **粗体**：`**粗体**` 或 `__粗体__`
- *斜体*：`*斜体*` 或 `_斜体_`
- ~~删除线~~：`~~删除线~~`
- `行内代码`：`` `代码` ``

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 列表

无序列表：

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

有序列表：

1. 第一项
2. 第二项
   1. 子项 2.1
   2. 子项 2.2

任务列表：

- [x] 已完成
- [ ] 待完成
- [ ] 进行中

## 代码块使用

### 插入代码块

1. 使用菜单：**格式 -> 代码块**
2. 或使用快捷键：`Ctrl + Shift + M`
3. 插入后会弹出语言选择器

### 选择语言

1. 在语言选择器的搜索框中输入语言名称（如 "python"）
2. 使用键盘上下键选择
3. 按回车键确认
4. 或直接点击所需语言

### 关闭选择器

按 `Escape` 键可关闭语言选择器。

### 修改语言

点击代码块，语言选择器会再次弹出，可重新选择语言。

## 常见问题

### Q: 打不开编辑器？

确保已正确安装 Rust 工具链和 Visual Studio Build Tools。详细请参阅开发者手册。

### Q: 文件保存失败？

检查目标文件夹是否有写入权限。

### Q: 中文显示异常？

编辑器支持 Unicode，请确保系统安装了相应的字体。

### Q: 代码块语言搜索不到？

语言选择器需要先在搜索框中输入文字才会显示匹配的语言列表。
